# MIT GSW Crowd Safety System

## Overview
2nd place at MIT Global Startup Workshop.  
Selected for MIT bootcamp.

## My Role
- Originated the idea
- Led system design
- Delivered final pitch

## Problem
High-density crowd environments collapse communication and create safety risks.

## Solution
A decentralized communication and crowd control system that works even when network infrastructure fails.

## System Design
- Peer-to-peer communication between devices  
- Real-time crowd density monitoring  
- Dynamic guidance to safer paths  

## Why It Matters
Existing systems fail when networks collapse.  
This approach maintains communication and improves safety in extreme conditions.
